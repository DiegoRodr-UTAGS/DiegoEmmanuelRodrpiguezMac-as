GitHub

¿Qué es GitHub?

GitHub es una plataforma de desarrollo colaborativo que utiliza Git como sistema de control de versiones. Es el lugar donde millones de desarrolladores y empresas construyen, mantienen y distribuyen software. GitHub permite alojar repositorios de código, colaborar con otros desarrolladores y realizar seguimiento de cambios en los proyectos.

La plataforma facilita el trabajo en equipo, ya que ofrece herramientas para gestionar versiones de software, revisar código, gestionar incidencias, y automatizar flujos de trabajo. GitHub también permite publicar proyectos de código abierto para que cualquiera pueda contribuir y aprender de otros.

¿Quiénes son los fundadores de GitHub?

GitHub fue fundada en el año 2008 por:

  • Tom Preston-Werner
  • Chris Wanstrath
  • PJ Hyett
  • Scott Chacon

Estos cuatro desarrolladores crearon GitHub con el objetivo de facilitar la colaboración en proyectos de software, especialmente aquellos que usan Git como sistema de control de versiones.
